# 🎮 Gamifica Mi Vida – API

🌐 Idioma: [English](README.md) | [Español](README.es.md)

API backend para un sistema personal de productividad y gamificación.

Organiza tareas, pendientes y recordatorios usando XP, estados y prioridades.

## 🚀 Tecnologías
- ASP.NET Core Web API (.NET 8)
- MySQL 8
- SQL directo (sin ORM)
- Arquitectura en capas

## ▶️ Ejecución
dotnet run --project src/GamificaMiVida.Api

## 📍 URL
http://localhost:5152
