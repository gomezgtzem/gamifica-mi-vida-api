# 🎮 Gamifica Mi Vida – API

🌐 Language: [English](README.md) | [Español](README.es.md)

Backend API for a personal productivity and life-gamification system.

This project manages tasks, reminders, and goals using gamification concepts such as XP points, completion status, and prioritization.

## 🚀 Tech Stack
- ASP.NET Core Web API (.NET 8)
- MySQL 8
- Raw SQL (no ORM)
- Layered architecture

## ▶️ Run
dotnet run --project src/GamificaMiVida.Api

## 📍 URL
http://localhost:5152
